# cell-aggregation-kinetics-openfoam
Cell Aggregation Kinetics Analysis and CFD Modeling of Spheroid Rounding

Description: This repository contains scripts for quantitative analysis and modeling of multicellular spheroid aggregation and shape dynamics. It includes:

ImageJ macros based on the SAMJ-IJ plugin for semi-automated segmentation of cell-filled microwells and time-lapse image analysis.

OpenFOAM launch files for CFD simulations of spheroid rounding in shaped microwells, treating spheroids as Newtonian fluids with surface tension.
